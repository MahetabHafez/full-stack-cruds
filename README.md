🚀 Full-Stack Product Management System (CRUDS)
A professional, high-performance web application built with the MERN Stack (MongoDB, Express, React, Node.js) to manage products efficiently. This project features a sleek Glassmorphism UI and complete CRUD functionality.

✨ Key Features
Create: Add products with title, price, taxes, ads, and category.

Bulk Add (Count): A unique feature to add multiple quantities of the same product at once.

Real-time Calculation: Automatic total price calculation including discounts.

Read & Search: View products in a structured table with real-time search by Title or Category.

Update: Edit existing product details with a smooth scroll-to-top experience.

Delete: Remove a single product or clear the entire database with a "Delete All" feature.

Responsive Design: Fully optimized for all screen sizes (Mobile, Tablet, Desktop).

🛠️ Tech Stack
Frontend: React.js, CSS3 (Advanced Glassmorphism), Axios.

Backend: Node.js, Express.js.

Database: MongoDB Atlas (Cloud Storage).

Deployment: Vercel (Frontend & Backend).

🎨 UI/UX Highlights
Glassmorphism Design: Modern transparent elements with background blur.

Interactive Elements: Smooth hover effects and transitions on all buttons.

Color Coding: - 🟢 Green: For safe actions (Update).

🔴 Dark Red: For destructive actions (Delete/Delete All).

🔵 Dark Blue: For primary actions (Create/Search).
